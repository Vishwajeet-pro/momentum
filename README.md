Momentum is a 5-page, fully responsive website for a modern, high-energy fitness studio. This project is built from scratch using only **HTML**, **CSS**, and **vanilla JavaScript**.

The design features a bold, dark-mode aesthetic with a striking "electric lime" accent color. It uses large, impactful typography ("Bebas Neue") to create a professional and motivating user experience.

## âœ¨ Features

* **5-Page Structure:**
    * **Home:** A full-screen hero section, "Why Join Us" icon grid, and "Class Spotlight" cards.
    * **Classes:** A complex, responsive **CSS Grid-based weekly schedule** for classes.
    * **Pricing:** A 3-column pricing table with a "popular" plan highlighted.
    * **About Us:** A 2-column section for the facility and a "Meet the Trainers" card grid.
    * **Contact Us:** A 2-column layout with a functional contact form and an embedded Google Map.
* **Fully Responsive Design:** Adapts cleanly to all screen sizes, from mobile phones to desktops, using modern CSS Flexbox, Grid, and media queries.
* **Modern CSS:**
    * Uses CSS Variables for an easy-to-manage color palette.
    * Strong font pairing (Bebas Neue & Montserrat) for a professional, bold look.
    * Clean, reusable card components.
* **JavaScript Functionality:**
    * **Responsive "Hamburger" Menu:** A mobile-friendly navigation that toggles on click.
    * **Form Validation:** Client-side validation for the "Contact Us" form to ensure all fields are filled.
